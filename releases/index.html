v2.1.1 (2019-10-25)

  1. In the previous version, the length of the sequence line in the fasta file should be 50 nt. If not, the result would be wrong. Here, in this versioin, the length is not limited.
  2. In the previous version, exons should be sorted by their chromosome position in the GTF file. If not, only few circRNAs could be found or only circRNAs in chromosome 1 (or 1 and 10) could be found. Here, the new version of UROBORUS will sort the GTF file before using it automatically, thus users can use any kinds of GTF files.
  3. In the previous version, the chromosome label should be 'chr1', 'chr2', 'chrX', .... If not, UROBORUS can not found any circRNAs. Here, in this version, the chromosome label can be '1', '2', 'X', ....
